## ✨ Features
- 🔐 JWT Authentication
- 🩸 Period Tracker with Predictions
- 😊 Mood & Symptoms Tracker
- 💧 Water Intake Tracker
- 📖 Health Blogs
- 👤 Profile Management
- 📊 Analytics Dashboard

## 🛠️ Tech Stack
**Frontend:** React.js, Tailwind CSS, Axios, React Router
**Backend:** Node.js, Express.js
**Database:** MongoDB Atlas
**Auth:** JWT, bcrypt

## 🏃 Run Locally

### Backend
cd server
npm install
npm run dev

### Frontend
cd client
npm install
npm run dev
